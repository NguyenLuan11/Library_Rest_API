# Library - API   v1.0

- API library have information about: books, author of book, category of book, students and list of borrow.
- API using Swagger UI.
- Json Web Token (JWT)

## Start

### Install
```bash
pip install -r requirements.txt
```

### Test
```bash
python -m unittest __file_name__.py
```

### Run
```bash
python app.py
```
